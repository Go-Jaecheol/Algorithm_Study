### Logic
* 첫번째와 두번째 문자열이 주어지면, 첫번째 문자열을 하나만 남기고 두번째 문자열을 하나씩 추가하면서 비교.
* 두번째 문자열을 끝까지 다 비교하면, 첫번째 문자열을 하나 추가.
* 이런식으로 첫번째 문자열의 끝까지 돌면, 가장 긴 문자열을 찾을수 있다.
