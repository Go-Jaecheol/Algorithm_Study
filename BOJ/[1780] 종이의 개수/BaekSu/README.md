## :round_pushpin: **Logic**

```c++
int check_paper(int x, int y, int size)
```

- 탐색할 종이의 크기가 **1x1인지, 1x1보다 큰 지** 확인
  - 1x1보다 큰 경우 종이 전체를 탐색하면서 하나의 숫자로만 이루어졌는지 확인
    - 하나의 숫자로만 이루어 진 경우 해당 숫자 종이 카운트
    - 그렇지 않은 경우에는 각 가로 세로를 1/3로 나누어 **재귀 호출**
  - 1x1인 경우는 해당 숫자 종이 카운트
