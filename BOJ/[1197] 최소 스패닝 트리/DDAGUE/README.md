### Logic
* cost 리스트에 노드와 간선을 받아서 저장함.
* 간선의 가중치를 기준으로 정렬함.
* while문 안에서 for문을 돌리고, set 알고리즘을 사용해 이미 갔던 노드는 걸러냄.
* set에 저장한 간선과 노드 집합은 -1로 값을 바꿔 다음 검색에 영향 없도록 해줌.
